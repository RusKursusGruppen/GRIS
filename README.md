# Conventions

Always write in English.
"Rus", "russer" and "rkg" are excepted from this

Translations
<Danish> = <English>
rus      = rus
russer   = russer
rkg      = rkg
vejleder = tutor
mentor   = mentor
rustur   = tour
hytte    = hut
